\section{Einführung}
	\subsection{Wissensbank}
		Eine Wissensbank speichert formalisiertes Wissen. Hat WR-Sprache mit Syntax und Semantik. terminologischen Informationen

	\subsection{Inferenzmaschine}
		Eine Inferenzmaschine leitet implizietes Wissen ab. Ist also ein Schlussfolgerungsverfahren.

	\subsection{Semantische Netze}
		Bestehen aus Knoten und Kanten. Knoten repräsentieren Individuen, Klassen und Eigenschaften. Kanten repräsentieren is-a und has-prop Beziehungen.\\
		\textbf{Problematik:}
		\begin{itemize}
			\item Mehrdeutigkeit von is-a
			\item Vererbung als default (keine terminologischen Infos)
			\item \glqq Semantischer Abstand\grqq{} abhängig von syntaktischer Details
			\item keine formale Semantik
			\item Netzwerk bedeuted was Ersteller meint
			\item Konflikte zwischen Benutzer und Programmierer nicht entscheidbar
		\end{itemize}

	\subsection{Frames}
		WR-Formalismus zum Beschreiben von Entitäten. Das Ziel war die Imitation von menschlichem Kategorisieren. Logisches Schließen sollte nicht möglich sein. Trennung von repräsntiertem Wissen und Schlussfolgerungsverfahren seien zu radikal. \\
		\textbf{Syntax:}
		\begin{description}
			\item[Class Frame] beschreibt Klasse von Entitäten
			\item[Instance Frame] beschreibt spezifische Entität 
			\item[slots] Frame enthält slots für bestimmte Attribute
			\item[filler] Wert eines slots wird filler genannt
			\item[Defaults] für filler sind möglich
			\item[Sub-frames] für Spezialisierung (is-a)
		\end{description}
		\textbf{Schlussfolgerungsverfahren:}
		\begin{description}
			\item[Vererbung] Sub-frames erben von Default-filler
			\item[Kriterialität] Wenn Instanz für alle Slots der Class Frame Y einen passenden Filler hat ist diese auch Instanz von Frame Y
			\item[Matching] Test ob Instanz eines Frames X auch Instanz des Frames Y ist 
		\end{description}
		\textbf{Probleme von Frames:}
		Frame-Inferenzen lassen sich in Prädikatenlogik ausdrücken und KI als Sammlung von Heuristiken.

	\subsection{Semantische Technologien}
		In anderen Worten \glqq Wann wird es Wahr?\grqq.

		\subsubsection{Syntax und Semantik}
			$4+=12($ ist syntaktisch falsch. \\
			$3 + 4 = 12$ ist syntaktisch richtig, aber semantisch falsch. \\
			$3 + 4 = 7$ ist syntaktisch und semantisch richtig.

		\subsubsection{Weitere Begriffe}
			Semantische Technologien sind Teilbereich der KI und beschreiben formale Beschreibung in einer Domäne.\\
			Semantic Web ist eine Erweiterung des WWW, die es ermöglicht, Daten mit Bedeutung zu versehen.\\
			Linked Data ist eine Methode, um Daten im Web zu verknüpfen.\\
			Ontologie ist die Lehre vom \glqq Seienden\grqq. Also unveränderliche und unvergängliche Ideen. Der Gegenbegriff wäre \glqq Werdendes\grqq. In der Informatik ist die Ontologie repräsentiertes Wissen. Beispielsweise Inhalt einer Wissensbank oder Instanzen von Klassen. \textcolor{red}{Ontologie besser beschreiben/definieren}\\
			Schlagwort-Verfahren bei beispielsweise Browsern kann funktionieren, muss aber nicht (Beispiel: Gorillas und Dunkelhäutige Menschen von Google KI). W3C hat Semantic Web Standards entwickelt.
		
	\section{Linked Data, URIs und RDF}
		\subsection{Linked Data}
			URIs sind Namen für Dinge.\\
			Verwende HTTP URIs, damit Menschen und Maschinen darauf zugreifen können.\\
			Wenn jemand eine URI nachschaut, sollte er nützliche Informationen erhalten die einen Standard verwenden.\\
			Verwende dort weitere URIs sodass Menschen weitere Informationen finden können.

		\subsection{URIs}
			Uniform Resource Identifier. Zeichenfolge, die eine abstrakte oder physische Ressource identifiziert.\\ URI ist ein Link, URL ist Teilmenge von URI und L steht für Locator. URN identifiziert Dinge (Menschen, Bücher, ...). URL liefert gesuchtes Objekt, URN liefert Datei die gesuchte Objekt beschreibt.

		\subsection{RDF}
			Resource Description Framework. Wird verwendet um Aussagen über Ressourcen zu machen. Besteht aus URIs, Literale und Leeren Knoten. URIs sind Namen für Ressourcen. Literale sind Werte. Leere Knoten sind anonyme Ressourcen.\\
			RDF-Graph ist eine Menge von Tripeln. Tripel besteht aus Subjekt, Prädikat und Objekt.\\
